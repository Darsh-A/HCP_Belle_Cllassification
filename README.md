## Info





### Event Type Flags:





$e^+ e^- \\to \\Upsilon (4S) \\to B^+ B^-$ = 0



$e^+ e^- \\to \\Upsilon(4S) \\to B^0 \\bar{B}^0$ = 1



$e^+ e^- \\to c \\bar{c}$ = 2



$e^+ e^- \\to u \\bar{u}$ = 3



$e^+ e^- \\to d \\bar{d}$ = 4



$e^+ e^- \\to s \\bar{s}$ = 5





For binary add (0 and 1) to one class and (2,3,4, and 5) to other class
